Email code examples

Thomson Reuters - event invitation

This event invitation email is based off of a template used for a while that was previously not responsive. I recoded the template so the right side column stacks underneath for mobile. For the numbered list, I used a table rather than ordered list tag since spacing can be more precisely controlled.

The email was deployed with Eloqua. There are personalization tags in the greeting for first name and in the footer for recipient email.
